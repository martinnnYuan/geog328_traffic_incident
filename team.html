Team & Approach

We pair SDOT collision data with traffic volume to find where risk is disproportionately high. Team: Martin Yuan, Jenny Liu, Xinrui Yang, Jiaxuan Su.

Data Sources

Traffic Flow Counts (2022) — AWDT/ADT from SDOT: data.seattle.gov
Collisions (All Years, filtered to 2022) — SDOT collisions: ArcGIS Open Data
Local copies live in assets/ for offline loading.

Processing

Client-side reprojection from EPSG:2926 to WGS84, then snapping collisions to nearest flow segment to compute crashes-per-volume risk.

Mapping

Custom Mapbox base map tinted for contrast; thematic layers are ordered above the base and below labels to keep text readable.

Next Steps

Add QA/QC on snapping radius, integrate a trusted segment ID join if available, and publish the map via GitHub Pages or Netlify.
